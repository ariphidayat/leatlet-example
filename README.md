## Leaflet Example Application using GeoJSON

### Technology Stack :
* [Leaflet](http://leafletjs.com/)
* [GeoJSON](http://geojson.org/)

### Prerequisite :
* Installed Node.js and npm

### Run and Build :
* Cxecute the following command :

     * npm install

     * grunt

* Call URL : [http://localhost:3000](http://localhost:3000)
